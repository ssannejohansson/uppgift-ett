Mina första dagar på IT: Tech for Women

De första lektionerna gick vi igenom grunderna, vilka jag var välbekant med sedan jag gjorde bloggdesigner i HTML som tonåring.
Del två av lektionerna handlade istället om Git och GitHub. Detta var helt nytt för mig och oj, vad jag fick kämpa. Det tog mig hela
dagen innan jag fick ordning på det, och när jag började om på nytt dagen därpå fick jag börja om igen då ingenting fungerade. Var
det inte fel på destinationen så var det fel på filerna, eller så gick det inte att få Git att samarbeta. Desperat laddade jag ned GitHub
desktop istället, men beslöt mig tillslut för att få ordning på allt genom terminalen innan jag började använda det. Nu, ca 17:30 dag
två har jag (nog) äntligen fått ordning på även detta och fått VS Code, GitHub Desktop och Git att samköra som de ska. Efter
måååånga om och men. Att vara envis och tycka att man kan lösa allt själv är verkligen en blessing in a curse, men jag tror att man lär
sig bäst genom att problemlösa på egen hand i den mån det går innan man blir fullkomligt tokig, haha.


Imorgon är det fredag och då ska jag göra de sista uppgifterna, spela Learning Git Branching och titta på tutorials på GitHub Skills.
Därefter ska jag ta helg och försöka smälta veckan, för att sedan se fram emot en ny vecka med nya utmaningar. Där jag förhoppningsvis
fortfarande är vän med Git...... annars får vi ta en vända till i ringen.
